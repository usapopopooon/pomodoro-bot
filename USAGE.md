# ポモドーロ Bot の使い方

このページは Discord で Bot を使う側の人向けです。
インストール / デプロイは [README.md](README.md) を見てください。

## 1 分ではじめる

1. 好きなテキストチャンネルで `/pomo` と入力
2. 出てきたパネルの **🙋 参加** を押す
3. **▶️ 開始** を押す(押した人が「オーナー」になります)
4. あとは 25 分作業 → 5 分休憩を繰り返すだけ

## パネルのボタン

### コントロールパネル(`/pomo` で出る常駐メッセージ)

```
[🙋 参加] [🚪 退出] [✍️ タスク] [📊 統計] [❓ 使い方]
[▶️ 開始]  [⚙️ 時間設定] [🔔 メンション] [🔊 ボイス] [🛑 終了]
```

**誰でも押せる(上段)**
| ボタン | 何をする |
|---|---|
| 🙋 参加 | このポモドーロに参加。別ルームに参加中なら自動で抜ける |
| 🚪 退出 | 抜ける |
| ✍️ タスク | 自分が今やってる作業を設定(他の人にも見える) |
| 📊 統計 | 自分の今日 / 今週 / 累計の 🍅 完了数を表示 |
| ❓ 使い方 | このパネルでヘルプを表示 |

**オーナー専用(下段)**
| ボタン | 何をする |
|---|---|
| ▶️ 開始 | タイマースタート(押すまでは設定変更可) |
| ⚙️ 時間設定 | 作業時間 / 休憩 / 長休憩の頻度を変更 |
| 🔔 メンション | フェーズ別(作業 / 短休憩 / 長休憩)にスポイラー付き mention の ON/OFF |
| 🔊 ボイス | オーナーが入っている VC に Bot を呼ぶ(再押下で抜ける) |
| 🛑 終了 | このポモドーロを終了 |

### フェーズパネル(各フェーズの境界で投稿される)

```
||<@111> <@222>||      ← メンション ON のフェーズだけ。クリックで展開
⏰ **作業**
████░░░░░░░░░░░░░░░░ 5分 / 25分
終了 <t:1735689600:R>   ← クライアント側でライブカウントダウン

[✅ 参加] [操作] [🛑 終了]
```

- **✅ 参加** — 途中参加もここからできる
- **操作** — 押すと一時停止 / スキップ / リセット / 時間設定 の ephemeral メニュー(オーナーのみ実行可、他は表示だけ)
- **🛑 終了** — オーナーがセッション終了

## 流れの例

### 自分ひとりで集中したい

1. `/pomo`
2. 🙋 参加
3. ✍️ タスクで今日やることを書く
4. ▶️ 開始

### 友達数人で集まって作業

1. 誰か 1 人が `/pomo`(その人がオーナー)
2. 全員 🙋 参加 して各自 ✍️ タスク を設定
3. オーナーが 🔔 メンション を ON にしておくと、休憩開始や作業開始のタイミングで全員に通知が飛ぶ(スポイラーで隠れるので画面はうるさくならない)
4. ▶️ 開始

### ボイスで合図を流したい

1. オーナーが Discord のボイスチャンネルに入る
2. コントロールパネルで **🔊 ボイス** を押す → Bot が同じ VC に来て「接続しました」
3. ▶️ 開始 で開始 / 各フェーズ境界で自動的に音声合図(後述)
4. もう一度 🔊 ボイス で Bot を VC から退出させられる

## 音声合図の中身

🔊 ボイス を ON にしたときだけ流れる音声(VC 接続中のみ)。
**音声は VOICEVOX:ずんだもん** で生成しています。アラーム音だけは効果音です。

| タイミング | ずんだもんが言うこと |
|---|---|
| 🔊 押下 | 「接続しました」 |
| ▶️ 開始 直後 | 「開始します」 |
| 作業終了 → 休憩開始 | 「ピンッ」(アラーム)+ 「休憩を開始します」 |
| 休憩終了 → 作業再開 | 「休憩を終了します」 |
| フェーズ終了 60 秒前 | 「あと 1 分です」 |
| ⏸ 一時停止 / 再開 | 「一時停止します」/「再開します」 |
| 🛑 終了 | 「終了します」 |
| 全員退出による自動終了 | (自動終了の合図) |

> **アラーム音は作業終了のときだけ** 鳴ります。休憩明けは静かに次に行くほうが集中の戻りが自然なので。

### クレジット

- **音声合成エンジン**: [VOICEVOX](https://voicevox.hiroshiba.jp/)(ヒホ)
- **キャラクター**: [VOICEVOX:ずんだもん](https://zunko.jp/)(東北ずん子・ずんだもんプロジェクト)

音声を再配布・再利用する場合は、上記 2 つを **「VOICEVOX:ずんだもん」** の形でクレジット表記してください(VOICEVOX 規約に従う)。本 Bot は非営利・個人利用の範囲で運用しています。

## こまかい挙動

### オーナーが抜けたら

- 残ってるメンバーで一番早く参加した人に自動的にオーナー権が移ります
- 全員抜けたらセッション自動終了

### 重複しない仕組み

- **1 チャンネル = 1 ポモドーロ** — 同じチャンネルで `/pomo` を再実行すると、古いセッションは静かに置き換わります
- **1 ユーザー = 1 セッション** — 別のチャンネルで 🙋 参加 すると、前にいたセッションから自動退出
- **1 サーバー = 1 VC** — Discord の制約で、同じサーバで Bot を 2 つの VC に同時に呼ぶことはできません(別の Bot を立てれば回避可能)

### 起動・再起動のとき

- Bot が再起動すると進行中のポモドーロは終了扱いになります(タイマーをメモリで持っているため)
- 古いパネルのボタンは「`/pomo` で作り直してください」という案内に置き換わるので、新しく `/pomo` を叩き直してください

## つまずきやすいところ

| 症状 | 確認すること |
|---|---|
| `/pomo` がスラッシュコマンド一覧に出てこない | グローバル同期は最大 1 時間。少し待つか、運営に開発サーバ ID を登録してもらう |
| ボタンを押すと「Interaction failed」 | Bot が直前に再起動した可能性。`/pomo` で作り直し |
| 🔊 押すと「先に VC に入ってください」と出る | Bot は **オーナー自身が入っている** VC に追従するので、先に VC に入ってから押してください |
| 🔊 押しても無音 | Bot に `Connect` / `Speak` 権限があるか確認(運営に依頼) |
| 通知が鳴らない | 🔔 メンション が ON か / Discord 側の通知設定が OFF になってないか |

## ヒント

- パネル下のフェーズメッセージに **クライアント側ライブカウントダウン** が出てるので、別タブで作業しながらでも残り時間が見える(`<t:UNIX:R>` 形式)
- 統計は 1 人 1 行で記録されるので、共同セッションでも各自の完了数が個別に積まれます
- 時間設定は **どのタイミングでも変更可** (実行中に変えると次のフェーズから反映、ラウンドカウントはリセット)
